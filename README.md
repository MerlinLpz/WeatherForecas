# WeatherForecast 🌤️

> ⚠️ This README is written in English as part of my language learning journey. English is my third language — after Spanish and Italian — and as a developer I know it's essential.

A SwiftUI weather forecast app built as part of my iOS development learning path.

## What does this app do?
Displays a 7-day weather forecast with:
- Individual day cards with weather icons and temperatures
- Weekly summary with sunny and rainy day count
- Average high and low temperature calculation
- Horizontal scroll for day-by-day forecast

## What I learned building this
- Computed properties in Swift
- View encapsulation with reusable components
- Connecting views by passing data
- `ForEach` and `ScrollView` in SwiftUI
- Git and GitHub from the terminal

## Tech Stack
- Swift
- SwiftUI
- Xcode 16
